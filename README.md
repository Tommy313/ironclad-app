# Ironclad Fleet Intelligence

Next.js fleet management application with AI-powered RAG chat panel.

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in your values
1. `npm install`
1. `npm run dev`

## Environment Variables

- `NEXT_PUBLIC_SUPABASE_URL` -- your Supabase project URL- `NEXT_PUBLIC_SUPABASE_ANON_KEY` -- your Supabase anon key
- `NEXT_PUBLIC_RAILWAY_URL` -- deployed Railway RAG backend URL
